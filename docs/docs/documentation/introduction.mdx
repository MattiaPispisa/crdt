# Introduction

This site documents **CRDT in Dart** — a workspace of small, composable
packages for building local-first, collaborative apps on top of
**Conflict-free Replicated Data Types**:

- **`crdt_lf`** — the core CRDT library (documents, handlers, changes,
  snapshots).
- **`crdt_lf_flutter`** — Flutter reactivity bindings.
- **`crdt_socket_sync`** — client/server synchronization over sockets, with a
  presence/awareness plugin.
- **`crdt_lf_sqlite` / `crdt_lf_drift` / `crdt_lf_hive`** — persistence
  adapters.
- **`hlc_dart`** — hybrid logical clock.

Each package's page under [Packages](/docs/packages) is its README — the
authoritative, always-up-to-date reference.

## New to CRDTs?

here: I wrote a post that introduces the topic from the ground up — the
problem they solve, the intuition behind them, and how the pieces fit
together.
